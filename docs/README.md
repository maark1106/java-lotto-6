로또 게임 기능 :

[x] 구입 금액 입력
    [x] 공백인지 검증
    [x] 정수인지 검증
    [x] 1000원 단위인지 검증
    [x] 최대 10억원까지만 가능

[x] 로또 정답 기능
    [x] 랜덤으로 6자리 발생

[x] 당첨 번호 입력
    [x] 공백이 있는지 검증
    [x]  각 자리가 정수인지 검증
    [x] 6자리인지 검증
    [x] 중복 검증
    [x] 1~45사이 수인지 검증

[x] 보너스 번호 입력 받기
    [x] 정수인지 검증
    [] 1~45사이 수인지 검증
    [] 당첨번호와 중복되지 않은지 검증

[] 구매 개수 출력
    [] 각 lotto 오름차순으로 정렬 후
    [] 로또 리스트 출력(보너스 번호 제외)

[] 당첨 결과 확인
    [] 각 로또들의 일치 개수 확인
    [] 각 등수에 대한 당첨 결과 출력
    [] 수익률 계산
